Before running the program, make sure to:
	- check mysql-connector-java.jar file is loaded to reference library
	- check execute query onto MySql workbench which found in 
		=> Servlet_Signup_Login/src/main/SQL/Servlet_Signup_Login query.sql
	- check for root password (String rpass;) inside the servlets and change its value to your own root password
	- then u r good to go.
